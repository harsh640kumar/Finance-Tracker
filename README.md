# 💰 Finance Tracker

A simple React-based finance tracking app to manage income and expenses.

## 🚀 Features
- Add / Edit / Delete transactions
- Track income & expenses
- View balance
- Clean UI

## 🛠 Tech Stack
- React
- Context API
- React Router
